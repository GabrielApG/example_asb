## Para executar a function:

- Versão do Node 16

### Recomendado uso do NVM

```
nvm install 16
nvm use 16  
```

### Verifique a versão instalada

```
node -v
```

### Atualize o projeto com 

```
npm install
```

### Execute o projeto

```
npm start
```

### Executar nestes unitários

```
npx mocha tests
```
